# LeadsMan - Google Business Scraper

## 📌 Project Overview
LeadsMan is a web-based application that scrapes business data from Google using Selenium and displays it in a structured format.

## 🚀 Features
- Scrape business data (name, phone, website)
- Export data to CSV
- Simple and user-friendly UI
- Works without API

## 🛠 Tech Stack
- Python
- Selenium
- FastAPI
- HTML, CSS, JavaScript

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Start backend:
   python backend/app.py

3. Open frontend:
   Open index.html in browser
